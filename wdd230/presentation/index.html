object-fit Property

The object-fit property defines how elements respond to the height and width of the container box it is placed in. It is intended to be used with images, videos, and other embeddable media formats in conjunction with the object-position property which we will not be getting into at this time. When object-fit is used by itself, it lets us crop an inline image by giving us fine-grained control over how it stretches and squishes inside its box or container. An inline image is a graphic image that is inserted into a web page to add visual appeal, provide information or to represent a hyperlink. Inline images are usually in GIF, JPEG or PNG format.


There are five different values you can use with the object-fit property:

  • fill
  • cover
  • none
  • contain
  • scale-down
