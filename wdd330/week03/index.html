Notes from Object methods, "this":

Method examples:

Object are usually created to represent entities of the real world.

  let user = {
    name: "John",
    age: 30,
  };

The first example that is used with the object user is as follows:
  let user = {
    name: "John",
    age: 30
  };

  user.sayHi = function() {
    alert("Hello!");
  };

  user.sayHi(); // Hello!

If there is only one object, you can use 'user'. If there are more than one object, you should to use 'this' instead or you will get an error.
  let user = {
    name: "John",
    age: 30,

    sayHi() {
    alert( user.name ); // leads to an error
    }

  };

  let admin = user;

  user = null; // overwrite to make things obvious

 admin.sayHi(); // TypeError: Cannot read property 'name' of null
If you were to replace user.name with this.name it would work just fine.
